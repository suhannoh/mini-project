# Mini Project – Backend

Spring Boot 기반 REST API 서버입니다.  
프론트엔드와 분리된 구조로
사용자 인증, 게시글 및 링크 관리 기능을 제공합니다.

## 주요 도메인
- auth
- user
- admin
- post
- link

JPA(Hibernate)를 사용하여 데이터베이스와 연동하며,  
환경별 설정을 통해 로컬/배포 환경을 구분하여 운영합니다.

## Tech Stack
- Java 17
- Spring Boot
- JPA (Hibernate)
- PostgreSQL
